# Active Directory Lab Series

> *Hands-on labs covering Active Directory domain setups, DNS configurations, Group Policy Object (GPO) administrations, and multi-domain controller replication architectures.*

| Lab | Topic | Description |
| :---: | :--- | :--- |
| **Lab 01** | [AD & DNS Baseline Setup](lab-01-ad-dns-setup.md) | Installing and provisioning Active Directory Domain Services (AD DS), domain controller DNS setups, and environment validation. |
| **Lab 02** | [GPO Fundamentals](lab-02-gpo-fundamentals.md) | Creating, linking, and managing Group Policy Objects (GPOs), working with policy inheritance, and setting system baselines. |
| **Lab 03** | [Advanced Group Policy Management](lab-03-advanced-gpo.md) | Configuring loopback processing, folder redirection, WMI filters, and registry preference policy overrides. |
| **Lab 04** | [Multi-DC Replication Architecture](lab-04-multi-dc-replication.md) | Setting up secondary domain controllers, managing Active Directory Sites and Services, and configuring cross-site replication topologies. |
